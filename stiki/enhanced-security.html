Enhanced Security Mode

To protect the privacy of our users we now offer the possibility of forcing all images to be loaded from a local SSL proxy to avoid leaking browsing history to other websites.

In addition to the added security, images will be also be pre-scaled to their thumbnail size before being downloaded, preventing annoying reflows during the loading of a thread.

Note: Enhanced security mode might make some threads take more time to load because the server has to fetch all the images in order to serve them from a trusted location.
